# FASTA Tools

A simple Python script using **Biopython** to analyze DNA sequences from a FASTA file.

## Features
- Reads sequences from a FASTA file
- Calculates:
  - Sequence length
  - GC content (%)

## Requirements
- Python 3.7+
- Biopython (`pip install biopython`)
